# winApi-MapTool
json과 winapi를 활용해 만든 간단한 맵툴
개발에 사용한 툴 : visual studio

2018년에 만든 winApi로 만든 mapTool

타이틀 화면에서 z키입력
메뉴 화면에서 숫자 1키 입력으로 맵툴
